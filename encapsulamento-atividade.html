Atividade Prática: Encapsulamento em Python

Programação Orientada a Objetos com foco em encapsulamento

Objetivo

Praticar conceitos de POO (Programação Orientada a Objetos) em Python, com foco em encapsulamento, classes, objetos e métodos get/set.

Exercício 1: Conta Bancária

Crie uma classe ContaBancaria que:

  • Atributos: cliente, num_conta, agencia, saldo
  • Encapsule o saldo (atributo privado)
  • Tenha métodos:
    • depositar(valor): aumenta o saldo se valor > 0
    • sacar(valor): diminui o saldo se valor <= saldo
    • get_saldo(): retorna o saldo atual

Exemplo de uso:

conta = ContaBancaria("João Silva", "12345-6", "001")
conta.depositar(100)
conta.sacar(30)
print(conta.get_saldo()) # Saída: 70

Exercício 2: Pessoa com Idade Controlada

Crie uma classe Pessoa que:

  • Atributos: nome, idade
  • Encapsule o atributo __idade (privado)
  • Tenha métodos:
    • set_idade(nova_idade): só aceita valores ≥ 0
    • get_idade(): retorna a idade
    • envelhecer(): aumenta a idade em 1 ano

Exemplo de uso:

p = Pessoa("Maria")
p.set_idade(25)
p.envelhecer()
print(p.get_idade()) # Saída: 26
p.set_idade(-5) # Deve exibir erro!

Exercício 3: Produto com Estoque

Crie uma classe Produto que:

  • Encapsule os atributos __nome, __preco e __estoque
  • Tenha métodos:
    • get_nome(), get_preco(), get_estoque()
    • set_preco(novo_preco): só aceita valores ≥ 0
    • adicionar_estoque(qtd): aumenta o estoque se qtd > 0
    • vender(qtd): diminui o estoque se qtd <= estoque

Exemplo de uso:

produto = Produto("Notebook", 4500, 10)
produto.adicionar_estoque(5)
produto.vender(3)
print(produto.get_estoque()) # Saída: 12

Exercício 4: Carro com Estado

Crie uma classe Carro que:

  • Atributos: marca, modelo, ano, ligado
  • Encapsule os atributos __ligado (bool) e __velocidade
  • Tenha métodos:
    • ligar(): só liga se não estiver ligado
    • desligar(): só desliga se velocidade == 0
    • acelerar(velocidade): aumenta a velocidade se o carro estiver ligado
    • get_status(): retorna "Ligado" ou "Desligado"

Exemplo de uso:

carro = Carro("Ford", "Fiesta", 2022)
carro.ligar()
carro.acelerar(50)
print(carro.get_status()) # Saída: "Ligado"
carro.desligar() # Só desliga se velocidade = 0

Como entregar?

  • Crie um arquivo .py para cada exercício ou um único arquivo com todas as classes
  • Teste cada método para garantir que as regras de encapsulamento estão funcionando

Desafio Extra (Opcional)

Modifique a classe Carro para incluir um limite de velocidade máxima (ex.: 120 km/h).
