🖥️ Projeto: Meu Portfólio HTML

Crie seu primeiro site completo com 5 páginas interligadas!

Início Exercícios Projeto Portfólio

📂 Estrutura do Projeto

Você vai criar um portfólio pessoal organizado da seguinte forma:

Estrutura de arquivos: 
📁 meu-portfolio/
├── 📁 css/
│   └── 📄 style.css       (Estilos do site)
├── 📁 imagens/
│   └── 🖼️ foto-perfil.jpg (Sua foto)
├── 📄 autoria.html          (Página Inicial)
├── 📄 sobre.html          (Sobre Mim)
├── 📄 interesses.html     (Meus Interesses)
├── 📄 projetos.html       (Meus Projetos)
└── 📄 contato.html        (Contato)
Dica: Crie essas pastas e arquivos antes de começar! Use o arquivo style.css para estilizar seu site.

🎯 Objetivo do Projeto

Você vai praticar tudo que aprendeu:

  • Links entre páginas - Criando um menu de navegação funcional
  • Âncoras - Para navegar dentro da mesma página
  • Elementos HTML - Títulos, parágrafos, listas, imagens
  • Organização - Estrutura de pastas e arquivos
Elementos que você pode usar:
  • <h1> a <h6> para títulos
  • <p> para parágrafos
  • <ul> e <ol> para listas
  • <img> para imagens
  • <a> para links e âncoras
  • <section> para organizar o conteúdo

1. Página Inicial (index.html)

Sua página principal deve conter:

  • Seu nome como título principal
  • Menu de navegação com links para todas as páginas
  • Sua foto (usando caminho relativo)
  • Uma breve apresentação sobre você
  • Lista de habilidades ou interesses principais
Dica de navegação: Todos os links do menu devem funcionar! Teste cada um deles.

2. Página "Sobre Mim" (sobre.html)

Nesta página, compartilhe:

  • Sua história e trajetória
  • Formação acadêmica (use listas)
  • Experiências relevantes
  • Link de volta para a página inicial
Dica: Use parágrafos (<p>) para textos longos e listas (<ul> ou <ol>) para itens.

3. Página "Interesses" (interesses.html)

Mostre seus gostos pessoais organizados em seções:

<h1>Meus Interesses</h1> <section id="livros"> <h2>📚 Livros Favoritos</h2> <ul> <li>Nome do Livro 1</li> <li>Nome do Livro 2</li> </ul> </section> <section id="musica"> <h2>🎵 Música</h2> <p>Meus artistas preferidos:</p> <ol> <li>Artista 1</li> <li>Artista 2</li> </ol> </section> <!-- Adicione mais seções conforme necessário -->
Dica de âncoras: Você pode criar links no topo da página para cada seção usando <a href="#livros">Livros</a>.

4. Página "Projetos" (projetos.html)

Apresente seus trabalhos:

  • Imagens dos projetos
  • Título para cada projeto
  • Descrição curta do que se trata
  • Tecnologias utilizadas (em lista)

5. Página "Contato" (contato.html)

Crie uma página com:

  • Seu email
  • Links para redes sociais
  • Âncora para voltar ao topo

📌 Conferindo

Antes de finalizar, verifique se:

  • Todas as páginas estão interligadas pelo menu
  • As imagens estão sendo exibidas corretamente
  • Os títulos estão hierarquizados (h1, h2, h3)
  • O conteúdo está organizado em seções
  • Todos os links funcionam
